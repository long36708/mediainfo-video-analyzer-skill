# Mediainfo Video 技能

## 概述

该技能能够使用 mediainfo.js 自动分析视频文件并提取全面的技术元数据,生成结构化的 .txt 报告。支持批量处理整个目录、递归扫描视频文件,并创建具有标准化命名规范的详细信息文档。

**核心功能:**
- 使用 mediainfo.js WebAssembly 库分析视频文件
- 提取技术规格:时长、格式、编码格式、分辨率、比特率
- 支持多种视频格式 (MP4、AVI、MOV、MKV、FLV、WMV、WebM 等)
- 支持递归目录扫描的批量处理
- 自动生成标准化的 .txt 报告,命名规则为:`{视频名称}_mediainfo_{时长}_{格式}__{编码}.txt`
- 详细的视频信息,包括音/视频轨道和元数据

## 使用方法

### 前置条件

确保 mediainfo.js 已正确安装并在项目中可用:

```bash
npm install mediainfo.js
```

### 处理视频

要分析目录中的视频文件并生成信息报告:

1. **定位包含视频文件的目标目录**
2. **确定输出目录** (如果未指定,则默认为与输入目录相同)
3. **使用相应参数执行处理脚本**

**命令语法:**
```bash
node scripts/process-videos.js <视频目录路径> [输出目录路径]
```

**示例 1: 在原处处理视频**
```bash
node scripts/process-videos.js /path/to/videos
```

**示例 2: 使用自定义输出目录处理视频**
```bash
node scripts/process-videos.js /path/to/videos /path/to/output
```

**示例 3: Windows 路径**
```bash
node scripts/process-videos.js C:/Videos C:/VideoInfo
```

### 输出文件命名

生成的 .txt 文件遵循以下命名规范:
```
{原视频名称}_mediainfo_{时长}_{格式}__{编码}.txt
```

**示例:**
- `myvideo_mediainfo_00_05_30_mp4__h264.txt`
- `movie_clip_mediainfo_01_23_45_mkv__hevc.txt`
- `recording_mediainfo_02_10_00_mp4__avc.txt`

### 支持的视频格式

脚本会自动检测并处理以下视频文件扩展名:
- `.mp4`, `.avi`, `.mov`, `.mkv`, `.flv`, `.wmv`, `.webm`
- `.m4v`, `.3gp`, `.ts`, `.mts`, `.m2ts`, `.vob`, `.ogv`
- `.asf`, `.rm`, `.rmvb`, `.divx`, `.xvid`, `.f4v`

### 报告内容

每个生成的 .txt 文件包含全面的视频信息,包括:
- 通用文件信息 (时长、格式、文件大小、比特率)
- 视频轨道详细信息 (编码格式、分辨率、帧率、宽高比)
- 音频轨道信息 (编码格式、采样率、声道数)
- 元数据标签 (如果可用)

### 工作流程

使用该技能时:

1. **定位处理脚本** 位于 `scripts/process-videos.js`
2. **执行脚本** 并指定目标视频目录路径
3. **监控输出** 以查看处理状态和结果
4. **查看生成的 .txt 文件** 其中包含详细的视频信息

脚本将:
- 递归扫描指定目录以查找视频文件
- 使用 mediainfo.js 分析每个视频文件
- 为每个视频生成一个 .txt 报告,并采用标准化命名
- 显示处理统计信息 (成功/失败计数)
- 优雅地处理错误并提供描述性消息

### 错误处理

脚本包含强大的错误处理机制:
- 验证输入目录是否存在
- 根据需要创建输出目录
- 为每个失败的视频报告具体错误
- 在完成时提供汇总统计信息
- 优雅地处理文件访问和解析错误

## 资源

### scripts/process-videos.js
编排视频文件分析的主要处理脚本。该 Node.js 脚本:

- 使用 mediainfo.js 以对象格式分析视频文件并提取元数据
- 提取时长、格式、编码格式、分辨率和其他技术细节
- 使用 mediainfo.js 以文本格式生成详细报告
- 生成标准化的 .txt 输出文件,并采用有序的命名
- 支持递归目录遍历的批量处理
- 提供显示进度和结果的详细控制台输出

**核心功能:**
- 通过扩展名自动检测视频文件
- 递归目录扫描以进行全面处理
- 标准化的输出文件命名规范
- 时长格式化 (基于长度使用 HH:MM:SS 或 MM:SS)
- 文件名中的特殊字符清理
- 全面的错误报告和统计

**使用方法:**
该脚本可以直接从命令行执行,也可以通过编程方式调用。它需要 Node.js 18+ 和已安装在项目中的 mediainfo.js。
